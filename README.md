#### 关键特性

* 16位ADC(3.6 MSPS)
* 2 x 12bit DAC
* 128 KB Flash + 1 MB SRAM

#### 封装规格

* TFBGA240 (14×14mm)
* UFBGA176 (10×10mm)
* LQFP176 (24×24mm)
* LQFP144 (20×20mm)
* LQFP100 (14×14mm)
